# Frontend for PostgreSQL Data Hub

This is a Next.js frontend for the PostgreSQL Data Hub API.

## Installation

```bash
npm install
```

## Configuration

Set the API URL in `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Build

```bash
npm run build
npm start
```
